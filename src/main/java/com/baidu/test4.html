上传常见问题

更多
1. 普通文档的上传量是怎样计算的?
普通文档的每日上传量由系统根据你的文档质量和重复度情况（考核指标达到若干个）来调整的，你当前评分为3.2分（总满分为 5 分），点击查看评分规则。新用户初次上传量为2000个。
2. 上传时提示标题太短或者标题重复是什么意思？
网站默认标题不能低于6个字，如你的正标题确实不足6个字，可以适当加一些关键词。
标题与别人上传的文档标题重复，但实际内容不重复也可进行适当修改、关键词换位，但必须通顺符合逻辑。切不可偷懒随意添加无关字符、删减标题，造成标题逻辑不通，违反网站规则。
3. 上传文档卡在待上传怎么办？
请更换浏览器重新上传，推荐使用360、chrome、firefox等浏览器。
4. 文档上传失败是什么原因造成的？
请检查网络，确认网络没有问题，请稍后刷新页面重试。
